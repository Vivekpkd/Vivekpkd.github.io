The Jump Mechanism

To jump to the application, we must: 1. Set the Main Stack Pointer (MSP) to the application's stack. 2. Set the Program Counter (PC) to the application's Reset Handler.

void JumpToApp(uint32_t appAddress) {
    uint32_t jumpAddress = *(__IO uint32_t*) (appAddress + 4);

    Jump();

Python Tutorial

Python is one of the most popular programming languages in the world. It is simple to use, packed with features, and supported by a wide range of libraries and frameworks. Its clean syntax makes it beginner-friendly and highly readable.

  • A high-level language used in data science, automation, AI, and web development.
  • Known for its readability — code is easier to write, understand, and maintain.
  • Backed by strong library support — we don't have to build everything from scratch.
  • Cross-platform — works on Windows, Mac, and Linux without major changes.

Variables and Data Types

Variables store data values. Python is dynamically typed — no need to declare a type explicitly.

# Variable examples
name = "Alice"
age = 25
pi = 3.14159
is_active = True

print(name, age, pi, is_active)

Output: Alice 25 3.14159 True

Operators

Python supports arithmetic, comparison, logical, and assignment operators.

a = 10
b = 3

print(a + b)   # Addition
print(a - b)   # Subtraction
print(a * b)   # Multiplication
print(a // b)  # Floor Division
print(a % b)   # Modulus
print(a ** b)  # Exponentiation

Output: 13 7 30 3 1 1000

Control Flow

if / elif / else

Use conditional statements to control the flow of execution.

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 75:
    print("Grade: B")
else:
    print("Grade: C")

Output: Grade: B

Loops

Python has for and while loops for iteration.

# for loop
for i in range(1, 6):
    print(i, end=" ")

Output: 1 2 3 4 5

# while loop
count = 0
while count < 3:
    print("Count:", count)
    count += 1

Output: Count: 0 Count: 1 Count: 2

Functions

Functions let you group reusable code under a name. Use def to define a function.

def greet(name, msg="Hello"):
    return f"{msg}, {name}!"

print(greet("Alice"))
print(greet("Bob", "Hi"))

Output: Hello, Alice! Hi, Bob!

Lambda Functions

A lambda is an anonymous, one-line function.

square = lambda x: x ** 2
numbers = [1, 2, 3, 4, 5]
result = list(map(square, numbers))
print(result)

Output: [1, 4, 9, 16, 25]

Data Structures

Python provides powerful built-in data structures out of the box.

Lists

Lists are ordered and mutable collections. They can hold mixed data types.

fruits = ["apple", "banana", "cherry"]
fruits.append("mango")
print(fruits)
print(fruits[1])

Output: ['apple', 'banana', 'cherry', 'mango'] banana

Tuples

Tuples are ordered and immutable — they cannot be changed after creation.

point = (10, 20)
print(point[0], point[1])

Output: 10 20

Dictionaries

Dictionaries store key-value pairs for fast lookups.

student = {"name": "Alice", "age": 22, "grade": "A"}
print(student["name"])
student["age"] = 23
print(student)

Output: Alice {'name': 'Alice', 'age': 23, 'grade': 'A'}

Sets

Sets are unordered collections of unique elements.

s = {1, 2, 3, 2, 1}
print(s)

Output: {1, 2, 3}

OOP Concepts

Python is a fully object-oriented language. Everything in Python is an object.

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} makes a sound."

class Dog(Animal):
    def speak(self):
        return f"{self.name} says: Woof!"

dog = Dog("Rex")
print(dog.speak())

Output: Rex says: Woof!

Exception Handling

Exceptions are runtime errors. Python lets you handle them gracefully using try / except.

try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"Error: {e}")
finally:
    print("Execution complete.")

Output: Error: division by zero Execution complete.

File Handling

Python can read from and write to files using the built-in open() function.

# Write to a file
with open("hello.txt", "w") as f:
    f.write("Hello, File!")

# Read from a file
with open("hello.txt", "r") as f:
    print(f.read())

Output: Hello, File!
